# 🚨 SafetyAlert - Women Safety Android App

📱 About the App
SafetyAlert is an Android application designed to enhance personal safety, especially for women. It provides quick emergency features that help users send alerts and share their location during dangerous situations.
🚀 Features
* 🆘 Emergency SOS button
* 📍 Live location sharing
* 📞 Quick call to emergency contacts
* 🔔 Instant alert notifications
* 👤 Simple and user-friendly interface
🛠 Tech Stack
* **Language:** Java / Kotlin
* **IDE:** Android Studio
* **Database:** Firebase / SQLite
* **Tools:** Google Maps API, Location Services
⚙️ Installation
1. Clone the repository:
```bash
git clone https://github.com/tejas20s/SafetyAlert.git
```
2. Open in Android Studio
3. Run the app on emulator or device
